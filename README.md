# 자바스크립트 알고리즘

## 목표

- README.md 파일에 사진과 코드를 기록으로 남기고, 배울 점이 있는 다른 사람들의 코드도 정리하여 알고리즘 실력을 향상시킨다.
- 클린 코딩을 생활화하고, 넓은 시야와 깊이 있는 학습을 목표한다.
- 매일 최소 한 문제 이상 해결을 목표로 한다.

- 문제 설명과 제한 조건이 기록된 이미지 파일을 첨부하고, 코드 또한 README.md 파일에 분류하여 기록
