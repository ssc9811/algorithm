## 약수의 합

### 문제 설명

```
정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.
```

### 제한 조건

- n은 0 이상 3000이하인 정수입니다.

### 입출력 예

| n   | return |
| --- | ------ |
| 12  | 28     |
| 5   | 6      |

### 코드

```javascript
function solution(n) {
  let sum = 0;
  for (let i = 1; i <= n; i++) {
    if (n % i === 0) {
      sum += i;
    }
  }
  return sum;
}
```

### 다른 사람의 풀이

```javascript
function solution(num) {
  let sum = 0;
  for (let i = 1; i <= num; i++) {
    if (num % i === 0) sum += i;
  }
  return sum;
}
```
