## 나머지가 1이 되는 수 찾기

### 문제 설명

```
자연수 n이 매개변수로 주어집니다.
n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요.
답이 항상 존재함은 증명될 수 있습니다.
```

### 제한 조건

- 3 ≤ n ≤ 1,000,000

### 입출력 예

| n   | return |
| --- | ------ |
| 10  | 3      |
| 12  | 11     |

### 내 풀이 코드

```javascript
function solution(n) {
  let answer = 0;
  for (let a = n - 1; a >= 2; a--) {
    if (n % a === 1) answer = a;
  }
  return answer;
}
```
